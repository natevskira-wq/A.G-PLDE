PLDE: Procedural Language Drill Engine
"Language is a physical reflex, not an intellectual puzzle."
PLDE is a high-intensity language training platform designed to build automaticity through procedural drills. It rejects gamification in favor of speed, strictness, and muscle memory.
Current Version: v5.1 (Director Mode & Smart Voice)

1. Prerequisites
You need a free Supabase project.
Create a project at supabase.com.
Go to Project Settings -> API and copy your URL and ANON KEY.
2. Database Schema
Run this SQL in your Supabase SQL Editor to set up the tables:
code
SQL
-- Smart Code User System
create table user_profiles (
  access_code text primary key,
  device_1 text,
  device_2 text,
  progress jsonb default '{}'::jsonb,
  created_at timestamptz default now()
);
alter table user_profiles enable row level security;
create policy "Public Access" on user_profiles for all using (true) with check (true);

-- Lesson Content System
create table lessons (
  id bigint generated by default as identity primary key,
  created_at timestamptz default now(),
  title text not null,
  content jsonb not null,
  sequence int default 999
);
alter table lessons enable row level security;
create policy "Public Read" on lessons for select using (true);
create policy "Public Write" on lessons for insert with check (true);
create policy "Public Update" on lessons for update using (true);
3. Storage Setup
Go to Storage in Supabase.
Create a new bucket named media.
Set it to Public.
4. Configuration
Open index.html and admin.html. Find the configuration block at the top and replace with your keys:

ðŸŽ® How to Use
For the Student (index.html)
Login: Enter an Access Code (issued by Admin).
Note: Codes are locked to the first 2 devices that use them.
Vector: Select your Native Language, Target Language, and Difficulty Level.
Drill:
Imprint: Listen and absorb.
Recognition: Rapid-fire identification (5s).
Construction: Reorder syntax (15s).
Production: Type or Speak the answer.
Recall: Final stress test.
Decay: Lessons not reviewed for 3 days will turn Yellow and lock new content until reviewed.
For the Admin (admin.html)
Create: Enter Trigger/Response pairs.
Direct: Use "Director Mode" to toggle Text/Translation visibility or set Voice Strictness (0.1 - 1.0).
Upload: Attach Audio/Images directly to the form.
Publish: One click saves the lesson to the Cloud and makes it live instantly.
