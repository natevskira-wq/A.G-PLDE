ðŸŽ® The Ecosystem
1. The Drill Engine (index.html)

Smart Code Login: 2-Device Lock. Auto-sync.

Director Mode: Supports "Audio Only" drills, "Test Mode" (Blind Imprint), and "Modular Loops" (Skip specific phases).

Decay Lock: Enforces maintenance. If a lesson decays (>3 days), new content locks.

2. The Creator Studio (admin.html)

Director Console: Toggle Text/Translation visibility, set Voice Strictness (0.1-1.0), and configure Auto-Play.

Private/Public: Creators can publish lessons visible only to themselves or to everyone.

AI Station: Paste prompts to auto-generate complex JSON structures.

3. The God Mode (manager.html)

User Management: Issue Access Codes in batches.

Licensing: Grant specific permissions (e.g., "Can teach German A1").

Device Reset: Clear stuck device locks for users.

ðŸ§  Advanced Logic

Universal Korean Matcher: Decomposes Hangul into Jamo to allow fuzzy matching for voice input.

Audio Matcher: A safety fallback for short words. Instead of speaking, the user identifies the correct sound from 3 options.

SRS Algorithm: Lesson review intervals expand exponentially (1 -> 3 -> 7 -> 14 -> 30 days) based on performance.
